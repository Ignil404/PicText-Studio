

## Конструктор картинок с текстом — Полная версия

### Страницы и навигация

**Главная** — галерея шаблонов с фильтрацией по категориям (мотивация, демотиваторы, открытки, поздравления, мемы, цитаты). Яркий градиентный хедер, поиск, карточки шаблонов с превью.

**Редактор** — основной экран работы:
- Canvas-превью изображения в реальном времени (HTML5 Canvas)
- Панель настроек: ввод текста (заголовок + подпись), выбор шрифта (8-10 вариантов Google Fonts), цвет текста (color picker), размер, позиция (drag или пресеты: верх/центр/низ)
- Кнопки: скачать PNG/JPEG, скопировать ссылку для шаринга
- Кнопка «AI-фон» — генерация фона по текстовому описанию через Lovable AI
- Загрузка своего фонового изображения

**Личный кабинет** — авторизация (email/Google), история созданных картинок, повторное редактирование и скачивание.

### 10-12 шаблонов на старте

По категориям: Мотивация (2-3), Демотиваторы (2), Открытки/поздравления (3), Мемы (2), Цитаты (2). Каждый шаблон — это набор параметров: фон (градиент/паттерн, сгенерированный через CSS), расположение текстовых зон, стиль шрифта, декоративные элементы.

### Дизайн — яркий и игривый

- Градиентные фоны (фиолетовый → розовый → оранжевый палитра)
- Округлые карточки, мягкие тени, крупные иконки
- Анимации при наведении, плавные переходы
- Весёлая типографика, emoji-акценты

### Техническая основа

- **Canvas**: рендер изображений через HTML5 Canvas для экспорта в PNG/JPEG
- **AI-фоны**: Edge function + Lovable AI (image generation) для создания уникальных фонов
- **Авторизация и хранение**: Lovable Cloud — аутентификация, хранение истории и сгенерированных изображений в Storage
- **Шаринг**: уникальные ссылки на созданные изображения
